Chapter 01

📦 변수 — 이름표가 붙은 상자

변수는 데이터를 저장하는 공간입니다. 상자 안에 값을 넣고, 그 상자에 이름표를 붙인다고 생각하세요.
이름표(변수명)를 부르면 언제든 상자 안의 값을 꺼낼 수 있어요.

⚙ 변수 만들기

위에서 변수를 추가하면 여기에 상자가 나타납니다 ✨
핵심 개념 — 변수를 만드는 것 = 이름표가 붙은 상자를 메모리(기억 공간)에 놓는 것. 값을 바꾸면 상자 안의 내용물만 교체됩니다.
variables.py 
# 변수 = 이름표 + 상자
name  = "홍길동"   # str 타입
age   = 17        # int 타입
score = 98.5     # float 타입
pass_ = True     # bool 타입

# 값 바꾸기 = 상자 안 내용물 교체
score = 100
print(score)  # 100

Chapter 02

🔢 1차원 리스트 — 나란히 놓인 상자들

리스트는 여러 값을 하나의 이름으로 묶어서 관리하는 자료구조입니다.
각 상자에는 인덱스(index)라는 번호가 붙어 있고, 0번부터 시작합니다.

⚙ 리스트 만들기

위에서 값을 입력하고 '시각화' 버튼을 눌러 보세요 ✨
인덱스는 0부터! — 리스트의 첫 번째 원소의 인덱스는 0입니다. 원소가 5개라면 인덱스는 0, 1, 2, 3, 4까지예요.
음수 인덱스 -1은 마지막 원소, -2는 뒤에서 두 번째를 가리킵니다.
list_1d.py 
# 리스트 만들기
scores = [85, 92, 78, 95, 88]

# 인덱스로 접근
print(scores[0])   # 85  (첫 번째)
print(scores[2])   # 78  (세 번째)
print(scores[-1])  # 88  (마지막)

# 값 변경
scores[1] = 100

# 추가 / 삭제
scores.append(70)   # 끝에 추가
scores.pop()         # 끝 원소 제거

# 길이
print(len(scores))   # 5

📊 2차원 리스트 — 격자(표) 형태

2차원 리스트는 리스트 안에 리스트를 넣은 구조입니다.
행(row)과 열(column) 두 개의 인덱스로 원소를 찾아요. 마치 엑셀 표처럼요!

⚙ 2차원 리스트 만들기

격자 설정 후 '시각화' 버튼을 눌러 보세요 ✨

현재 리스트 코드
접근 방법: 리스트[행][열] — 먼저 몇 번째 행인지, 그다음 몇 번째 열인지 순서대로 씁니다. 예: matrix[1][2]는 2행(인덱스 1) 3열(인덱스 2)의 값이에요.
list_2d.py 
# 2차원 리스트 = 리스트의 리스트
matrix = [
    [1, 2, 3],   # 0행
    [4, 5, 6],   # 1행
    [7, 8, 9],   # 2행
]

# [행][열] 순서로 접근
print(matrix[0][0])  # 1  (0행 0열)
print(matrix[1][2])  # 6  (1행 2열)
print(matrix[2][1])  # 8  (2행 1열)

# 전체 순회
for row in matrix:
    for val in row:
        print(val, end=" ")
    print()  # 줄바꿈

함수의 핵심 3요소 — ① 함수 이름 (호출할 때 사용), ② 매개변수 (입력값을 받는 변수), ③ return (결과를 돌려주는 키워드). return 없이 끝나면 자동으로 None을 반환합니다.
function_basics.py 
# 함수 정의
def add(a, b):
    return a + b

# 함수 호출 — 소괄호 안에 인수(argument)를 넣는다

print(result)   # 7

# 기본값 매개변수
def greet(name, age=17):
    return f"안녕하세요, {name}! {age}살이군요!"

print(greet("민수"))       # age=17 기본값 사용
print(greet("지영", 19))  # age=19 지정

📊 2차원 — 표 형태

행 × 열 구조
[0][0]1
[0][1]2
[1][0]4
[1][1]5
행·열 두 인덱스로 접근
구분 변수 1차원 리스트 2차원 리스트
저장하는 값 값 1개 값 여러 개 (한 줄) 값 여러 개 (행×열)
접근 방법 x x[i] x[i][j]
인덱스 개수 0개 1개 2개 (행, 열)
비유 상자 1개 서랍 한 줄 엑셀 스프레드시트
주요 활용 단일 값 저장 성적, 목록 등 행렬, 지도, 좌석표
자주 하는 실수 — 리스트 인덱스를 1부터 세는 실수! 파이썬은 무조건 0부터 시작합니다. 원소가 5개인 리스트는 인덱스 0 ~ 4, 5번 인덱스를 쓰면 오류가 납니다.
summary.py 
# ─── 변수 ───
age = 17
print(age)           # 17

# ─── 1차원 리스트 ───
scores = [85, 92, 78]
print(scores[0])    # 85
print(len(scores))  # 3

# ─── 2차원 리스트 ───
seats = [
    ["A1", "A2", "A3"],
    ["B1", "B2", "B3"],
]
print(seats[0][2])  # "A3"  (0행 2열)
print(seats[1][0])  # "B1"  (1행 0열)

# 행의 수, 열의 수
print(len(seats))      # 2  (행)
print(len(seats[0]))  # 3  (열)
